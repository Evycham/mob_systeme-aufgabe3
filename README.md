# Downloader (Android)
Downloader für Android. Kurzes Android-Projekt (Kotlin, Gradle Kotlin DSL) für die dritte Aufgabe in Mobile Systeme.

## Features
- URL-Eingabe mit Validierung
- Download im Hintergrund (Foreground Service)
- Live-Fortschritt in der UI
- Status-Feedback bei Erfolg/Fehler

## Voraussetzungen
1. Android Studio (aktuelle Version)
2. Android SDK minSdk 33, targetSdk 36


## Starten
1. Projekt in Android Studio öffnen.
2. App auf Emulator oder Gerät ausführen.
3. URL eingeben und `Download starten` tippen.
